SOBRE MI

Soy estudiante de Ingeniería en Sistemas de Información (actualmente cursando tercer año) y me apasiona la programación y el desarrollo de soluciones web.

Al principio me interesé más por el Desarrollo Web Frontend, pero, a pesar de gustarme, me sentía de cierta forma incompleto porque lo que pensaba o ideaba desarrollar no lo podía cubrir sólo con esos conocimientos. Entonces, supe que había un mundo más allá del Frontend y me incliné por conocerlo.

Crear aplicaciones web completas, seguras y eficientes es algo que me motiva mucho y por ello empecé a darme cuenta que el Desarrollo Web Full Stack es lo que realmente me entusiasma.

Me encantan los desafíos y las oportunidades que me permitan aplicar mi creatividad y seguir creciendo.

¡Gracias por visitar mi espacio digital!

Julio 2024 - Enero 2025

Peluquería Invasión

Desarrollador Full Stack

Participé en el desarrollo de una aplicación web para la gestión integral de una peluquería. Incluye funcionalidades como la gestión de turnos, el seguimiento de ingresos mediante un registro de caja, la administración de empleados y servicios ofrecidos, y el cálculo automático de pagos a cada empleado según porcentajes acordados.

El backend fue desarrollado con Node.js, Express y JWT para autenticación segura, utilizando una arquitectura de API REST. La base de datos, alojada en Turso, fue mapeada con Drizzle ORM y SQLite. El frontend se creó con HTML, CSS, JavaScript y Vite. Proyecto desplegado en Vercel.

HTML CSS JavaScript NodeJS Express JWT SQLite Drizzle ORM API REST
